# AMR 17 Motorsport - Generador de Presupuestos

Proyecto: generador de presupuestos para AMR 17 Motorsport.

Cómo ejecutar en local:

1. Abrir terminal en la carpeta del proyecto.
2. Iniciar servidor HTTP (puerto 5500):

```bash
python -m http.server 5500 --bind 127.0.0.1
```

3. Abrir en el navegador: `http://127.0.0.1:5500/final_v8_fixed.html`
